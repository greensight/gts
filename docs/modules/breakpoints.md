# Модуль: `breakpointsFromTokenManager`

Извлекает брейкпоинты из **grid-стилей** (`styles.grid`): числовые ключи сортируются по возрастанию и сопоставляются с именами из `names` (с конца массива имён к большим значениям ключей).

**Исходный код:** `src/modules/breakpoints/breakpointsFromTokenManager/module.ts`

## `input`

| Поле | Тип | Обязательность | По умолчанию | Описание |
|------|-----|----------------|--------------|----------|
| `names` | `string[] \| undefined` | Опционально | `['xxxl', 'xxl', 'xl', 'lg', 'md', 'sm', 'xs', 'xxs', 'xxxs']` | Имена брейкпоинтов. Количество имён должно быть **не меньше**, чем число числовых ключей в grid-данных; иначе ошибка. |

**Ограничения:**

- Нужен загруженный `TokenManager` и наличие `styles.grid`.
- Все ключи уровня брейкпоинтов в grid должны быть **числовыми** (строки, парсящиеся в число). Нечисловые ключи приводят к ошибке.

## `output`

| Поле | Тип | Обязательность | Описание |
|------|-----|----------------|----------|
| `dir` | `string` | **Обязательно** | Каталог для сгенерированных файлов брейкпоинтов. |

## Пример

```typescript
breakpointsFromTokenManager({
    input: { names: ['xl', 'lg', 'md', 'sm', 'xs'] },
    output: { dir: './dist/breakpoints' },
})
```
